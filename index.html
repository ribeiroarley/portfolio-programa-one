Challenge Número Secreto

O Jogo do Número Secreto é um desafio em que você deve adivinhar um número aleatório gerado pelo sistema. A cada tentativa, o sistema informará se o número secreto é maior ou menor do que o seu palpite.

Challenge Amigo Secreto

Uma aplicação web simples e divertida para organizar sorteios de Amigo Secreto entre amigos ou familiares!

Jogo do Número Secreto
Exemplo de Uso:
  • Abra o jogo em seu navegador.
  • Digite um número entre 1 e 1000 no campo de entrada.
  • Clique no botão "Chutar" ou pressione Enter para fazer sua tentativa.
  • O sistema exibirá se o número secreto é maior ou menor do que o seu chute.
  • Continue tentando até acertar, e então clique em "Reiniciar" ou aguarde 15 segundos para começar uma nova rodada.
